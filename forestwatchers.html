ForestWatchers.net a citizen project for forest monitoring

ForestWatchers.net is a citizen cyberscience project where volunteers from all over the world can help in assessing deforestation from satellite images. The project currently have two running web applications:

  • Best Tile. Volunteers help selecting the less cloudy image for the same are from satellite images.
  • Deforested Areas. Volunteers help to identify deforested areas (even in indigenous or federal conservational areas) by drawing polygons.
